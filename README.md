# morr
A CSS Starting Point


### Installation

`npm install morr`  

### Usage

Font: `<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Raleway:300,600">`  

Include `morr.min.css` in your setup and you're good to go.
